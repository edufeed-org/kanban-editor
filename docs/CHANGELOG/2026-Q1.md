Stand: 2026-02-20  
Quelle: `docs/CHANGELOG/archive-legacy-2026-02-20.md` + Git PR-Merges

## Kurzfazit

Q1/2026 war kein einzelner Release, sondern ein schneller Stabilitaets- und Ausbau-Zyklus mit vier Schwerpunkten:

1. Kollaboration und Sync stabilisiert (Shared Boards, Patches, Tombstones, Guards).
2. UX deutlich verbessert (Inline Editing, Mobile/UI-Polish, Navigation/Follow).
3. AI- und Content-Workflows ausgebaut (populate/paste/OER).
4. Betrieb und Sicherheit gehaertet (OIDC Fixes, Docker Publishing, Event-Authorisierung).
