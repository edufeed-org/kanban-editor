# 📚 Benutzerhandbuch: KI-Kanban-Board für Lehrkräfte

**Willkommen beim EduFeed Kanban-Board!**

Dieses Handbuch hilft Ihnen, das KI-gestützte Kanban-Board optimal für Ihre Unterrichtsplanung und Materialorganisation zu nutzen.

---

## 🎯 Was ist das Kanban-Board?

Das EduFeed Kanban-Board ist ein intelligentes Planungswerkzeug speziell für Lehrkräfte:

- **📋 Visuelle Organisation:** Planen Sie Unterrichtseinheiten, Projekte und Aufgaben übersichtlich
- **🤖 KI-Unterstützung:** Lassen Sie sich von der KI beim Strukturieren und Recherchieren helfen
- **🔍 OER-Materialien:** Finden Sie passende Open Educational Resources direkt im Board
- **👥 Kollaboration:** Arbeiten Sie gemeinsam mit Kolleg:innen an Boards
- **🌐 Dezentral:** Ihre Daten gehören Ihnen – gespeichert über das Nostr-Netzwerk

## 🚀 Schnellstart in 3 Schritten

### 1. Board öffnen
Besuchen Sie die Anwendung – Sie können sofort mit einem **Demo-Board** starten, ohne Anmeldung!

### 2. Erste Karte erstellen
Klicken Sie auf **„+ Karte hinzufügen"** in einer Spalte und geben Sie einen Titel ein.

### 3. KI fragen
Öffnen Sie den **KI-Assistenten** (rechte Sidebar) und fragen Sie z.B.:
> „Erstelle mir 5 Karten für eine Unterrichtseinheit zum Thema Photosynthese"

**Das war's!** Die KI erstellt automatisch strukturierte Karten für Sie.

## ❓ Häufige Fragen

### Muss ich mich anmelden?
**Nein!** Sie können sofort mit einem Demo-Board starten. Für dauerhafte Speicherung und Kollaboration empfehlen wir eine Anmeldung über Nostr (NIP-07 Browser-Extension).

### Wo werden meine Daten gespeichert?
- **Lokal:** Im Browser (IndexedDB) – funktioniert auch offline!
- **Dezentral:** Optional über Nostr-Relays (Ihre Wahl der Server)
- **Nie:** Auf zentralen Servern, die wir kontrollieren

### Ist das kostenlos?
**Ja!** Das Projekt ist Open Source und wird durch Fördermittel finanziert.

### Brauche ich einen OpenAI-Account für die KI?
Für die KI-Funktionen wird ein OpenAI API-Key benötigt. Diesen können Sie in den Einstellungen hinterlegen. Ohne API-Key funktionieren alle anderen Funktionen problemlos.
