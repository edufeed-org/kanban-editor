# 01 - Schnellstart: In 5 Minuten zum ersten Board

**Ziel:** Nach diesem Kapitel haben Sie Ihr erstes Board mit Karten erstellt und wissen, wie Sie die wichtigsten Funktionen nutzen.

---

## 🎬 Los geht's!

### Schritt 1: Anwendung öffnen

Öffnen Sie die Kanban-Board Anwendung in Ihrem Browser. Sie sehen sofort:

```
┌─────────────────────────────────────────────────────────────┐
│  📋 Kanban Board                         [Einstellungen] 👤 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  📂 Meine Boards          │    Willkommen!                  │
│  ─────────────────        │                                 │
│  + Neues Board            │    Wählen Sie ein Board oder    │
│                           │    erstellen Sie ein neues.     │
│  🎓 Demo Board            │                                 │
│                           │                                 │
└─────────────────────────────────────────────────────────────┘
```

### Schritt 2: Demo-Board ausprobieren (empfohlen für den Start)

Klicken Sie auf **🎓 Demo Board** – Sie erhalten ein vorkonfiguriertes Board mit Beispiel-Karten:

| Spalte | Inhalt |
|--------|--------|
| **📥 Ideen** | Gesammelte Unterrichtsideen |
| **🔄 In Arbeit** | Aktuell bearbeitete Materialien |
| **✅ Fertig** | Abgeschlossene Einheiten |

**💡 Tipp:** Das Demo-Board ist perfekt zum Ausprobieren – Sie können nichts kaputt machen!

### Schritt 3: Erste Karte erstellen

1. Gehen Sie zur Spalte **📥 Ideen**
2. Klicken Sie auf **+ Karte hinzufügen**
3. Geben Sie einen Titel ein, z.B.: `Einführung Photosynthese`
4. Drücken Sie **Enter** oder klicken Sie auf **Erstellen**

**🎉 Geschafft!** Ihre erste Karte ist da.

### Schritt 4: Karte verschieben (Drag & Drop)

1. Klicken Sie auf Ihre neue Karte und **halten Sie die Maustaste gedrückt**
2. Ziehen Sie die Karte in eine andere Spalte (z.B. **🔄 In Arbeit**)
3. Lassen Sie los

Die Karte ist jetzt in der neuen Spalte – so einfach organisieren Sie Ihren Workflow!

### Schritt 5: Karte bearbeiten

1. **Klicken** Sie auf eine Karte
2. Es öffnet sich das **Bearbeitungsfenster**:

```
┌─────────────────────────────────────────────────────────────┐
│  Einführung Photosynthese                              [X]  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Beschreibung:                                              │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ Die Schüler lernen die Grundlagen der              │   │
│  │ Photosynthese kennen...                            │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  Labels: [Biologie] [Klasse 7] [+ Label]                   │
│                                                             │
│  🖼️ Bild hinzufügen    🔗 Link hinzufügen                  │
│                                                             │
│                                    [Abbrechen] [Speichern]  │
└─────────────────────────────────────────────────────────────┘
```

3. Fügen Sie eine **Beschreibung** hinzu
4. Vergeben Sie **Labels** zur Kategorisierung
5. Klicken Sie auf **Speichern**

## ❓ Häufige Fragen zum Start

### Wird mein Demo-Board gespeichert?
Ja! Das Demo-Board wird im Browser gespeichert und ist beim nächsten Besuch noch da. Für permanente Speicherung empfehlen wir eine Anmeldung.

### Kann ich das Demo-Board löschen?
Ja, über die Board-Einstellungen. Bei Anmeldung wird es automatisch zu einem normalen Board konvertiert.

### Die KI antwortet nicht?
Für KI-Funktionen benötigen Sie einen OpenAI API-Key. Diesen können Sie in **Einstellungen → KI-Konfiguration** hinterlegen.
